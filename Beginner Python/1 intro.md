# Introduction to Python

## What is Python?
- **Python** is a **high-level, simple, and easy-to-learn programming language**.
- It is **free and open-source** and **portable** across platforms.
- Developed by **Guido van Rossum**.

**Example Program**
```python
print("Hello World")
```
Python Fundamentals
Character Set
Letters: A–Z, a–z

Digits: 0–9

Special symbols: + - * / % etc.

Whitespaces: space, tab, newline

Supports ASCII and Unicode characters

Variables
Variables are names given to memory locations.

```py
name = "Archit"
age = 23
price = 25.99
```
Identifiers (Naming Rules)
Can contain letters, digits, and underscore (_).

Cannot start with a digit.

Cannot use special symbols like !, #, @, %, $.

Can be of any length.

Data Types
int – Integer values

float – Decimal numbers

str – Strings

bool – True/False

complex – Complex numbers

Other Concepts
Keywords
Reserved words such as:
if, else, for, while, def, class, True, False, None

Comments
# Single-line comment

```py
"""
Multi-line
comment
"""
```

Operators
Arithmetic: + - * / % **

Comparison: == != > < >= <=

Assignment: = += -= *= /=

Logical: and, or, not

Type Conversion and Casting
Type Conversion: Automatic conversion by Python

Type Casting: Manual conversion using functions
```py
int("10")
float("3.14")
str(25)
#Input from User
name = input("Enter name: ")
age = int(input("Enter age: "))
price = float(input("Enter price: "))
```
